About this vault

A quiet place to file away the how-tos, shortcuts, and small fixes worth remembering — so future-you never has to re-learn them the hard way.

Why this exists

Every time you figure something out — a keyboard shortcut, a build step, a fix that took twenty minutes of searching — it's easy to forget it a month later. This site is a personal catalog for those moments: short, searchable entries you can find again in seconds.

How it works

Entries are organized like index cards, filed under a category and tagged with keywords. Use the search bar and filters on the home page to find what you need. If editing is unlocked, you can add, edit, or delete any entry directly from the page — no code required.

Built simply, on purpose

No backend, no database, no accounts. Everything runs in plain HTML, CSS, and JavaScript, and entries are saved locally in your browser. That keeps it fast, private, and easy to host anywhere.

Filed & kept for future-you. Everything here lives in your browser's local storage — no server, no database.
